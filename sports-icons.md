## Instructions to Update:

1. Go to Sports Management page
2. Click Edit (pencil icon) for each sport
3. Paste the corresponding icon URL
4. Click "Update Sport"

## Notes:
- Iconify CDN is recommended as it's reliable, fast, and free
- You can change the color by modifying the `color` parameter
- You can adjust size with `width` and `height` parameters
- All URLs work without authentication or API keys
